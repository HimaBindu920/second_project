CodinGame is a technology company editing an online platform for developers, allowing them to play with programming with increasingly difficult puzzles, to learn to code better with an online programming application supporting twenty-five programming languages, and to compete in multiplayer programming contests involving timed artificial intelligence, or code golf challenges.

CLASH OF CODE: CHALLENGE YOUR FRIENDS TO SHORT CODING BATTLES

Clash of Code is a multiplayer mode where you can challenge your friends in short real-time coding battles.

Once you start a clash, the type of clash is chosen randomly between these three types:

Fastest: easy, you just have to be the first to pass all the tests*

Shortest: code golf! Pass all the tests* with the minimum amount of characters. Ruby, Python, and JavaScript are your new friends.

Reverse: the statement is hidden; only test cases (input + expected output) are given. You should guess what to do. The fastest to pass the tests* wins.



Clash of Code Live Events

There will also be Code Live Events where you can exhibit your coding skills.
